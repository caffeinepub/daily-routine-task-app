# Trusted Web Activity (TWA) Setup Guide

This guide explains how to package the Daily Routine Task App as an Android application for Play Store publishing using Bubblewrap.

## Prerequisites

1. **Node.js** (v14 or higher)
2. **Java Development Kit (JDK)** 8 or higher
3. **Android SDK** with Build Tools
4. **Bubblewrap CLI**
